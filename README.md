## ⚡ Core Modules

### 01_BOOT // The First Impression
A cinematic CRT-style boot sequence that initializes system protocols, verifies identity, and transitions seamlessly into the main interface. Powered by **GSAP** and custom CSS shaders.

### 02_SHELL // The Interactive Workspace
A high-performance "Glassmorphic" window management system allows concurrent exploration of my Bio, Technical Skills, and the System Terminal.

### 03_VAULT // The 3D Project Archive
An immersive, **Three.js** driven 3D grid where projects aren't just links—they are holographic artifacts. Experience full interactivity with orbiting particles and dynamic lighting.

### 04_MATRIX // The AI Concierge
A **LangChain** powered AI interface integrated directly into the OS. It "reads" the portfolio's knowledge base to answer questions about my experience, tech stack, and availability in real-time.

## 🏗 Project Structure

```text
NEXUS/
├── frontend/             # Next.js 14 Application (OS Shell & UI)
│   ├── src/components/   # Boot, Desktop, and Window components
│   ├── src/context/      # Global OS & Window state management
│   └── src/hooks/        # Animation & UI orchestrators
├── backend/              # Node.js + TS (AI Agent & RAG Service)
└── data/content/         # Source Knowledge Base for the AI
```
